
# Yah Wright Playlist #1

This is a modular version of your playlist web app.

## Files Included

- index.html → Main HTML structure
- style.css → Custom styling
- script.js → Playlist logic and audio controls
- README.md → Project documentation

## How To Use

1. Unzip the folder.
2. Open index.html in your browser.
3. Ensure internet connection for Tailwind & Google Fonts CDN.

## Customization

You can:
- Add more songs inside script.js
- Replace audio URLs
- Host on any static web server (Netlify, Vercel, GitHub Pages, etc.)
